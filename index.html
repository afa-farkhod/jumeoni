Translation is solved. Understanding isn't.

Papago and Naver can translate the words. But translated text is still incomprehensible if the system is foreign:

📬

"Is this a fine or junk mail?"

You get an official-looking envelope. Translating it doesn't tell you whether to act, panic, or recycle.

💊

"Can I drink with this?"

The pharmacy hands you 4 pills. The label has the drug name in Korean. Side effects? Interactions? Good luck.

🏠

"What did my landlord just say?"

A KakaoTalk message arrives. Is it urgent? Are you in trouble? How do you reply politely?

📑

"Am I about to get deported?"

An immigration letter. Translated, it's still bureaucratese. You need someone to say "renew by March or else."

How Jumeoni works

1

Drop the document

Photo, screenshot, or PDF — any Korean paperwork. Drag it in or click to upload.

2

AI reads and reasons

Claude's vision model identifies the document type, extracts deadlines and amounts, and judges urgency.

3

You get a plan, not a translation

What it is. What to do. By when. Plus next-step actions like adding to calendar or drafting a Korean reply.

⚙ Configure API key Click to set up

Jumeoni runs entirely in your browser. To call Claude, paste your Gemini API key. It's stored in localStorage only — never sent anywhere except https://ai.google.dev/gemini-api/docs/api-key.

Production note: For a real deployment, proxy through a backend (Cloudflare Worker, Vercel function) so users don't need their own key. See README for the proxy pattern.
